# PhoneUSG™ — AI Stethoscope in Your Pocket

> **₹40 piezo sensor + phone mic → Detects heart murmurs in 30 seconds**  
> **Offline. Open-source. Works in rural clinics.**

---

## Features
- Record heart sounds via mic or external probe
- AI classifies: **Normal / Murmur / S3 / S4**
- Fully offline (TensorFlow Lite)
- Hindi + English UI
- Share PDF report
